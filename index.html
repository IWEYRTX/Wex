Наши проекты

Wexium Linux

это open source дистрибутив, основанный на Linux. Он предназначен для использования на настольных компьютерах. Wexium Linux предлагает широкий спектр функций и возможностей, включая поддержку различных графических оболочек и программного обеспечения.

Wexium ARM

это версия Wexium Linux, оптимизированная для работы на устройствах на базе процессоров ARM. Она предназначена для использования на мобильных устройствах.
